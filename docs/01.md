# แบบจำลอง deep learning พอสังเขป

พิจารณาปัญหาการจำแนกข้อมูล ถ้าให้ $$\mathcal{X}$$ เป็น input space เราสามารถมองแบบจำลอง deep learning เป็นฟังก์ชัน $$h_\theta : \mathcal{X}\rightarrow \mathbb{R}^k$$ ที่รับข้อมูลจาก $$\mathcal{X}$$ และให้ผลเป็นเวกเตอร์ของจำนวนจริง $$k$$ มิติ เมื่อ $$k$$ เป็นจำนวนของชนิดข้อมูลที่ต้องการจำแนก ตัวอย่างเช่น หากเราต้องการจำแนกภาพสีขนาด $$244\times 244$$ พิกเซล ออกเป็น 1000 ชนิด เราสามารถสร้างแบบจำลองให้รับ input $$x$$ ใด ๆ เป็นเวกเตอร์ขนาด $$244\times 244\times 3$$ มิติ โดยที่สมาชิกแสดงความเข้มของสีแดง / เขียว / น้ำเงิน ของแต่ละพิกเซล และให้ผลลัพธ์เป็นเวกเตอร์ 1000 มิติ ($$k=1000$$)

# Linear binary classification

หลังจากที่เราได้รู้จักหลักในการสร้าง adversarial example และการเทรน robust model กันแล้ว เราจะลองนำหลักดังกล่าวมาวิเคราะห์บนแบบจำลอง classification ที่ง่ายที่สุดก่อน นั่นคือแบบจำลอง logistic regression ซึ่งจะใช้ฟังก์ชันเชิงเส้น (linear function) ในการแบ่งข้อมูลออกเป็นสองกลุ่ม เราสามารถแทนแบบจำลองได้ด้วยฟังก์ชันดังนี้

$$
h_\theta(x)=w^Tx+b
$$

เมื่อ $$\theta=\{w\in\mathbb{R}^n, b\in\mathbb{R}\}$$ โดยมี class label $$y\in\{+1,-1\}$$ 

สำหรับ loss function ในกรณีที่เราต้องการจำแนกข้อมูลเป็นสองกลุ่ม (กลุ่มที่ 1 และกลุ่มที่ 2) หากเราใช้แบบจำลองที่คืนค่า $$h_\theta(x)$$ เป็นเวกเตอร์สองมิติ และใช้ฟังก์ชัน softmax ในการคำนวณความน่าจะเป็นที่ตัวอย่างข้อมูล $$x$$ ใด ๆ จะอยู่ในแต่ละกลุ่ม จะได้ว่าความน่าจะเป็นที่ $$x$$ จะอยู่ในกลุ่มที่ 1 มีค่าเป็น

$$
\frac{e^{h_\theta(x)_1}}{e^{h_\theta(x)_1}+e^{h_\theta(x)_2}}
= \frac{1}{1+e^{h_\theta(x)_2-h_\theta(x)_1}}
$$

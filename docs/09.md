# Projected Gradient Descent

จากการสร้าง adversarial example ด้วย FGSM สังเกตว่า หากลักษณะความโค้งของ loss function เมื่อเทียบกับ input นั้นไม่เป็น convex 
จุดที่มีค่า loss สูงสุดในระยะ $$\epsilon$$ จาก $$x$$ อาจไม่ใช่จุดมุมที่มีทิศทางตรงกับ gradient ของ $$x$$ ก็ได้ เนื่องจากทิศทางของ gradient ก็สามารถเปลี่ยนไปได้เมื่อเราเปลี่ยนตำแหน่งไปจากเดิม แม้ว่าจะห่างจากเดิมน้อยมาก
